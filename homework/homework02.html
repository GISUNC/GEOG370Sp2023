Homework 2: GitHub Page

Deadline:

hw2

Instructions: On Github, make a page that has a map in PNG format, links to vectors in GeoJSON format and a link to one vector data source obtained from the internet.

  • The map should contain a vector obtained from the internet and a background raster (it can be a tile layer).
  • The map should also have a point, a line and a polygon digitized by the student in WGS84. The point, line and polygon should have a readable attribute displayed in the map.
  • The map should also have a title and a legend.
  • Export the map as a PNG.
  • Watch the tutorial videos on how to create a GitHub page, and how to digitize a point, a line and a polygon and export them as GeoJSON.
- -

What will you turn in before the deadline?

  1. Upload the link to the HTML page in Github that contains the map, a link to the location where you obtained the vector from the internet, and a link to your folder in GitHub where you have uploaded the point, line and polygon in GeoJSON format.
